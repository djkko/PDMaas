# 1. 项目介绍 | Intro

PDMaas是一款优秀的国产数据建模工具，所有操作系统都能用，主打轻便好看效率高。它能帮你：

1. 拖拽画数据表和ER关系图，概念/逻辑/物理三层建模  
2. 把流程图、思维导图整合到一起看  
3. 主流数据库（如MySQL、PostgreSQL）、国产数据库、大数据MPP库全兼容  
4. 有标准字段库和枚举代码库  
5. 自动生成Word、HTML、Markdown文档和Java/Python等代码  
6. 模型和数据库双向同步，自动生成DDL差异脚本  
7. 支持多版本管理，便捷追溯模型历史变更  

- **核心定位**：社区开源版（CE）面向开发者，支持全流程数据模型设计与管理。   
- **适用场景**：开发者、技术爱好者

### 发展历史  

2018年初，团队针对传统建模工具局限，仅用一个半月完成PDMan 1.0开发，实现产品从无到有突破；同年5月发布首个开源版本，开启开源探索之路，持续更新至2019年1月。因前期设计存在局限，升级难度增大，2019年底团队启动系统性规划设计。

2022年4月，融合PDMan核心标识与实体关系图（ER）概念的PDManer 4.0正式上线，全面承接并超越过往版本功能，迅速跻身国内开源数据库工具领先行列。

随着数据要素市场加速构建及AI技术迅猛兴起，数据的核心价值愈发凸显，企业对数据建模在大规模协作、标准化规范及在线服务化部署等方面的需求持续升级，PDManer架构逐渐难以满足需求。

2023年团队开启商业化探索，基于PDManer推出企业级协作产品PDMaas。2024年7月，第二代PDMaas发布，完成架构全面升级，**确立PDMaas为统一品牌，划分版本**：
- **PDMaas-CE (社区单机版)**：免费开源社区版，适用于技术爱好者；
- **PDMaas-Pro (高级单机版)**：高级功能单机版，满足专业个人需求；
- **PDMaas-EE (企业级Web版)**：功能全集 + 企业级协作平台，面向企业及团队用户。

此次升级完整承接原PDManer 98%以上功能，成功实现从开源社区工具到企业级全方位数据建模解决方案的转型。
**PDMaas将成为本产品长期使用的唯一名称，不再进行任何变更。**

如同红军→八路军→解放军的历史演进，正是PDMan→PDManer→PDMaas产品迭代的最佳类比：不同历史阶段承担不同使命，核心团队始终如一的坚守，以及能力与定位的持续性升级，共同推动着产品不断进化。
